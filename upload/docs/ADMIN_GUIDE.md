# Admin Usage Guide — Dynamic Segmentation

### Creating Segmentation Rules
1. Go to **Rules** tab.
2. Click **Add Rule (+)**.
3. **Thresholds:** Set spend (USD) and order frequency thresholds.
4. **Demographics:** Set age range and gender.
5. **Timeframe:** 
   - *Calendar Month*: Resets on the 1st of every month.
   - *Rolling Window*: Continuously checks the last X days.
   - *Custom*: Fixed date range (e.g., for a Black Friday campaign).
6. **Notification:** Enable push notifications and enter the message title/body to be sent when a user joins this segment.

### Personalized Display
1. **Banners Tab:** Assign banners to specific customer groups.
2. **Product Sliders Tab:** Choose products to display for specific groups.
3. **Promotions Tab:** Map your existing coupons/promotions to these segments.

### Manual Overrides
If you need to keep a specific customer in a group (e.g., a whitelist or an VIP VIP), go to **Manual Assign**, search for the customer, and toggle the **Lock** icon. This prevents the segmentation engine from moving them automatically.
